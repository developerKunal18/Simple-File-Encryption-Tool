# Simple File Encryption Tool

A Python script to encrypt and decrypt text files.

## Description
This project uses a basic encryption technique (character shifting) to secure file content and restore it later.

## Features
- Encrypt file content
- Decrypt file content
- Simple key-based system
- Beginner-friendly encryption logic

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Place a text file in the project folder
3. Run:

   python encryptor.py

4. Choose encrypt/decrypt
5. Enter file name and key
